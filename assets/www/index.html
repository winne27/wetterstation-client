Datenschutz
Diese App erhebt keinerlei persönliche Daten. Es wird lediglich eine anonyme Zugriffsstatistik erzeugt. Protokolliert werden: Name der abgerufenen Datei, Datum und Uhrzeit des Abrufs, übertragene Datenmenge, Meldung über erfolgreichen Abruf. Zusätzlich werden die IP Adressen der anfragenden Rechner protokolliert. Diese App ermittelt zwar den Standort, diese Information wird aber nur zur Entfernungsbestimmung genutzt und nicht auf einen Server übertragen.
Beschreibung
Hier werden die Messdaten einer privaten Wetterstation in Rhauderfehn-Burlage in Ostfriesland angezeigt. Die Daten werden automatisch aktualisiert.

Durch Links-Rechts-Wischgesten kann zwischen den Aktuellen Messdaten, der Tagesübersicht, der Monatsübersicht und der Jahresübersicht gewechselt werden.

Der Sonnenschein wird über die Leistung und den Ertrag einer Photovoltaikanlage mit 7,15kWp gemessen.

Diese Informationen können auch im Web unter der Adresse www.fehngarten.de abgerufen werden.
